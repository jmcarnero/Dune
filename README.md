# Dune v1.1 [José M. Carnero](https://sargazos.net)
===

## PHP framework.

- Ligero.
- MVC.
- Desacoplado.
- Servicio REST.
- Sistema de URL's limpias, independente de *mod_rewrite*.
- Traducción automática de textos.

###Librerías incluidas:

- Captchas.
- Envío de correos.
- Feeds RSS
- Formularios.
- Listados.
- Miniaturas de imágenes.
- Sistema unificado de acceso a datos: SQL, CSV
